$p.local

5.0_5.5005A.20230904.185702

HTML5 LocalStorage를 사용하여 구현된 기능입니다.
LocalStorage는 도메인(사이트)별로 브라우저에서 임의의 영역에 데이터를 저장하는 기능으로, 브라우저 설정에 따라 저장할 수 있는 데이터 양이 결정됩니다.
IE 하위 버전도 지원합니다. (세부 내용 및 스펙은 http://www.w3.org/TR/webstorage/#storage 참조하세요.)
쿠키와 비슷하지만 만료일이 없고, 브라우저에 따라 저장 가능한 용량이 5MB ~ 10MB 사이입니다.
멀티 브라우징을 위해 5MB 이하로 저장하는 것이 권장됩니다.

Method Detail

clear()

동일한 도메인으로 localStorage에 저장 된 모든 데이터를 삭제합니다.
이 함수는 'Page Scope'이 적용되지 않습니다.

Sample

// localStorage에 저장 된 모든 데이터를 삭제합니다. $p.local.clear();
getAllItem()

동일한 도메인으로 localStorage에 저장 된 모든 데이터를 key,value형태의 객체로 반환합니다.

Return
  • Type
    JSON
  • Description
    key,value가 저장 된 JSON 객체.

Sample

var tmpData = $p.local.getAllItem(); //return 예시 ) { name:"WebSquare" , add:"서울시" }
getItem(keyName)

동일한 도메인의 LocalStorage에서 첫 번째 인자(keyName)에 할당된 값을 반환합니다.
이 함수는 'Page Scope'이 적용되지 않습니다.

Parameter
  • Name
    keyName
  • Type
    String
  • Required
    Y
  • Description

    LocalStorage에 저장된 KEY

Return
  • Type
    String
  • Description
    첫 번째 인자(keyName)에 할당된 데이터가 있으면 값을 반환하며, 그렇지 않을 경우 null을 반환합니다.

Sample

// 예시 1) 할당된 값이 있는 경우 - $p.local.setItem("exam_1", "WebSquare"); 또는 localStorage.setItem("exam_1","WebSquare"); var strResult1 = $p.local.getItem("exam_1"); // return 예시) "WebSquare" // 예시 2) 할당된 값이 없는 경우 var strResult2 = $p.local.getItem("exam_2"); // return 예시) null
key(index)

동일한 도메인의 LocalStorage에서 Index에 해당하는 KEY를 반환합니다.
LocalStorage에 저장된 KEY의 Index 값은 브라우저마다 다를 수 있습니다. 예를 들어 Chrome은 마지막에 저장한 값이 0번째에 위치하지만 IE는 첫 번째에 저장한 값이 0번째에 위치합니다.
'$p.local.length' 함수와 함께 사용하여 전체 KEY 목록을 확인할 수 있습니다.
이 함수는 'Page Scope'이 적용되지 않습니다.

Parameter
  • Name
    index
  • Type
    Number
  • Required
    Y
  • Description

    저장된 KEY의 Index.

Return
  • Type
    String
  • Description
    첫 번째 인자(index)에 해당하는 KEY 값을 반환하며, 값이 없는 경우에는 null을 반환합니다.

Sample

// 아래의 스크립트를 통해 1개의 값이 LocalStorage에 저장되어 있는 경우 $p.local.setItem("exam_1", "WebSquare"); // 예시 1) var strResult = $p.local.key(0); // return 예시) 'exam_1'
removeItem(key)

동일한 도메인의 LocalStorage에서 첫 번째 인자(key)와 일치하는 데이터를 삭제합니다.
모든 데이터를 삭제하려면 '$p.local.clear()'를 사용합니다.
이 기능은 'Page Scope'이 적용되지 않습니다.

Parameter
  • Name
    key
  • Type
    String
  • Required
    Y
  • Description

    데이터(item)의 KEY

Sample

// 아래의 스크립트를 통해 1개의 값이 LocalStorage에 저장되어 있는 경우 $p.local.setItem("exam_1", "WebSquare"); // 예시 1) Local 'exam_1'로 저장된 데이터를 삭제합니다. $p.local.removeItem("exam_1");
