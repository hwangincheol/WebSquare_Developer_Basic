WebSquare.uiplugin.switch

5.0_5.5005A.20230904.185702

스위치. 각 상황 별 화면을 동적으로 표시. 동적인 화면 전환을 지원. 즉, 여러 상황에 대한 복수의 Group들이 겹쳐 존재하고, 각 상황에 대한 Group을 동적으로 표시. 기본적으로 case1만이 존재하고, case 추가가 가능. 여러 레이어가 존재하나 조건에 따라 하나만 보여줄 때 사용. 탭과 비슷하게 동작하지만, 화면에 표시할 case는 API를 통해 설정 가능.
