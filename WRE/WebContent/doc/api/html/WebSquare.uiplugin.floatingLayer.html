WebSquare.uiplugin.floatingLayer

5.0_5.5005A.20230904.185702

플로팅 레이어. 팝업 형태의 화면을 생성. 일반 팝업과의 차이는 팝업을 띄울 화면에 컨텐츠를 미리 배치시키고 show/hide API를 이용하여 숨기거나 보일 수 있음. Group 컴포넌트와 같은 구조로 컴포넌트 하위(안쪽)로 직접 컴포넌트를 배치하거나 WFrame 또는 IFrame 컴포넌트를 배치시켜 화면을 링크. 일반적으로 body 하위에 배치하며 본 컴포넌트의 최상위는 <div>로 하위에 복합 태그를 포함.
