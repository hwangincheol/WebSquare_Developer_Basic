WebSquare.uiplugin.iframe

5.0_5.5005A.20230904.185702

아이 프레임. HTML의 IFrame과 유사. IFrame을 통해 WebSquare 파일, 일반 웹 컨텐츠, 다른 사이트들을 화면에 출력 가능.
SPA 모드 적용이 가능한 컴포넌트이며. 단, SPA 적용을 위한 제약사항을 반드시 준수해야 함.

Property Detail

allowTransparency

IE만 지원. 컴포넌트에 투명 배경 적용 여부. [true, false]

class

HTML의 class 속성과 동일한 기능을 제공. CSS 파일이나 style 블럭에 정의한 여러 개의 class를 공백으로 구분하여 적용 가능.
Engine 내부에서 각 컴포넌트 별로 지정된 class를 적용하나, class 속성을 이용하여 컴포넌트의 CSS를 제어 가능.

id

컴포넌트 ID.
컴포넌트 ID는 전역 객체로 할당됨. 각 컴포넌트 ID를 통해 Script에서 해당 컴포넌트에 접근 가능.

name

HTML의 name 속성과 동일한 기능을 제공.
Anchor 등에서 target으로 설정 가능.

relativePath

XML이 아닌 다른 주소(예 :.html)에 대해서도 src 상대 경로를 지원하는 속성. [default: false, true]
XML은 기본적으로 상대 경로를 지원하나, 그 외에는 상대 경로를 지원하지 않음.
하위 호환 유지를 위한 속성.

scrolling

src 화면의 width/height가 IFrame의 width/height 보다 클 때 스크롤 바를 표시 여부. [auto, no, yes]

spa

SPA 모드 적용 여부. [default:false, true]
src 화면이 웹스퀘어 화면인 경우만 해당.
SPA 모드는 IFrame 내의 웹스퀘어 자원들을 재사용.
src 화면이 동적으로 변경될 경우 해당 화면의 소스와 필수 리소스만 로딩하여 적용.

config.xml 파일을 통한 공통 설정 <spa value="true"/>

spaReplaceHistory

SPA 사용 시 IFrame 내부의 history(앞으로 가기 및 뒤로 가기) 동작을 금지. [default: false, true]
IFrame 내부 페이지의 뒤로 가기 동작을 막아서 blank 페이지로 이동하지 않는 효과.
이 경우, 뒤로 가기를 수행 할 경우 브라우저 전체가 뒤로 이동.

(옵션 설명) "true" : 뒤로 가기 및 앞으로 가기 동작을 금지. "false" (기본 값): 동작을 허용.

src

iframe이 표시할 화면 URL.

title

HTML의 title 속성과 동일.
마우스 오버 시, title로 저장한 값이 툴팁으로 표시됨. 일반적으로 웹접근성 지원용으로 설정.

tooltip

사용자가 컴포넌트에 마우스-오버할 때 표시되는 텍스트.
웹스퀘어가 제공하는 툴팁으로, 컴포넌트의 추가 정보를 명시.

tooltipLocaleRef

tooltip 문구가 정의되어 있을 경우 클라이언트 다국어 적용시 툴팁에 출력하는 텍스트의 key.
key에 해당하는 텍스트가 없을 경우 "global_undefined"가 화면에 표시됨.
