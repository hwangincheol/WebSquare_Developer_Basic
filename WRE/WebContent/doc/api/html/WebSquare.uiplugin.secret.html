WebSquare.uiplugin.secret

시크릿. 사용자 입력을 "*"로 표시. 사용자 입력을 "*"로 표시하는 InputBox 형태의 컴포넌트. HTML <input> 태그의 type=password와 유사한 인터페이스를 제공. InputBox에 비해 축소된 기능을 제공.
