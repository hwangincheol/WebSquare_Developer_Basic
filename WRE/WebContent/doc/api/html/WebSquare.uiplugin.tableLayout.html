WebSquare.uiplugin.tableLayout

5.0_5.5005A.20230904.185702

테이블 레이아웃. <table>로 레이아웃을 구성. TableLayout을 사용하여 전체 레이아웃을 구성하는 것은 비권장. 컴포너트 및 텍스트를 배치할 때 유용하며, 주로 입력용 또는 검색용 화면을 구성할 때 사용함. 최상위는 <table> 태그로 구성.
