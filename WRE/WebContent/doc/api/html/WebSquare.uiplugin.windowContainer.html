WebSquare.uiplugin.windowContainer

5.0_5.5005A.20230904.185702

윈도우 컨테이너. MDI 지원. 여러 화면들을 여러 윈도우에 수평배열, 수직배열, 순차배열, 균등배열 방식으로 배치. 일반적으로 MDI(multiple document interface)라고 표현.
최상위는 HTML <div>로 구성.
