첫번째 인자로 주어진 Document 또는 Element를 기준으로 XPath로 조회한 node에 attribute나 text노드를 추가합니다.

노드를 생성하지 못하게 되면 아무런 작업을 수행하지 않습니다.(에러도 발생하지 않습니다.)

  • Required
    Y

  • Name
    value1
  • Type
    String
  • Required
    Y
  • Description

    값을 설정할 attribute 이름. value2가 지정되어 있지 않으면 Text 노드 값으로 사용된다.

  • Name
    value2
  • Type
    String
  • Required
    Y
  • Description
