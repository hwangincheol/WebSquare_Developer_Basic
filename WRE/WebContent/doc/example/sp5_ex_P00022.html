컴포넌트 ID 문자열로 컴포넌트 객체 반환받기

1개요

$p.getComponentById 예제입니다. 컴포넌트의 ID 문자열을 받아 웹스퀘어 컴포넌트 객체를 반환합니다.

이 기능은 주로 공통 스크립트를 작성할 때 사용됩니다.

P00022

2구현된 기능

  • 컴포넌트의 ID로 컴포넌트 객체 반환받기

3예제 테스트 방법

  1. 컴포넌트 ID 항목에 ibx_ex1를 입력합니다.

  2. 컴포넌트 Value 항목에 friday를 입력합니다.

  3. 버튼 컴포넌트 ID 문자열로 컴포넌트의 value 할당하기를 클릭합니다.

  4. ID : ibx_ex1 항목의 inputbox에 friday가 출력됩니다.

4구현 예시

• 컴포넌트를 찾지 못한 경우는 undefined이 반환됩니다.

var objCmp;

//컴포넌트의 ID 속성값이 "ibx_ex1"인 컴포넌트를 반환받습니다.
objCmp = $p.getComponentById("ibx_ex1");

//반환된 객체가 없는 경우. - 메시지 처리 및 로직 중단.
//반환된 객체가 없는 경우 undefined 이 반환됩니다.
if (!objCmp) {
    alert("정의된 컴포넌트를 찾을 수 없습니다.");
    return;
}

//ibx_ex1 컴포넌트의 value 속성에 friday를 할당합니다.
objCmp.setValue("friday");

5주요 API

$p.getComponentById( )
