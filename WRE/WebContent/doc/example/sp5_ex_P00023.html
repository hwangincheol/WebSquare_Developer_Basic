[컴포넌트 공통] 컴포넌트의 이름(pluginName) 반환받기

1개요

컴포넌트의 이름(pluginName)을 반환 받는 예제입니다. 컴포넌트의 getPluginName method를 사용합니다.

P00023

2구현된 기능

  • InputBox 컴포넌트의 이름 반환받기

  • InputCalendar 컴포넌트의 이름 반환받기

  • Checkbox 컴포넌트의 이름 반환받기

3예제 테스트 방법

InputBox의 컴포넌트 이름 반환받기

1. InputBox영역의 버튼 실행1 을 클릭합니다.

2. alert 창에 input이 출력됩니다.

InputCalendar의 컴포넌트 이름 반환받기

1. InputCalendar영역의 버튼 실행2 를 클릭합니다.

2. alert 창에 inputCalendar 가 출력됩니다.

Checkbox의 컴포넌트 이름 반환받기

1. Checkbox영역의 버튼 실행2 를 클릭합니다.

2. alert 창에 checkbox 가 출력됩니다.

4주요 API

  • getPluginName( )

5구현 예시

var strCmpType;

//ibx_ex1 컴포넌트의 plugin name 반환 받기
strCmpType = ibx_ex1.getPluginName();
