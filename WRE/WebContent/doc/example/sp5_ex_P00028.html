[컴포넌트 공통] 스크립트로 style 제어하기

1개요

컴포넌트의 style을 스크립트로 변경하는 예제입니다.
아래의 API로 CSS 속성을 제어 할 수 있습니다.
- CSS 속성값 설정 : setStyle
- CSS 속성값 반환 : getStyle
- 초기 CSS 속성값으로 복구 : setInitStyle
- 초기 CSS 속성값 반환 : getInitStyle

P00028

2구현된 기능

  • InputBox의 border-color 변경 하기

  • InputBox의 border-color 속성값 반환 받기

  • InputBox의 color 속성값을 복구하기

  • InputBox의 초기 style 값 반환 받기 - border-color

4구현 예시

4.1InputBox의 border-color 변경 하기

//id가 ibx_input인 컴포넌트의 예시입니다.

//border-color를 #e40a0a로 할당합니다.
ibx_input.setStyle("border-color", "#e40a0a");

4.2InputBox의 border-color 속성값 반환 받기

//id가 ibx_input인 컴포넌트의 예시입니다.

var strRet;

//inputbox의 border-color style 값 반환받기.
strRet = ibx_input.getStyle("border-color");

4.3InputBox의 border-color 속성값을 복구하기

//id가 ibx_input인 컴포넌트의 예시입니다.

//컴포넌트의 초기 스타일로 복구합니다.
ibx_input.setInitStyle("border-color");

4.4InputBox의 초기 style 값 반환 받기 - border-color

//id가 ibx_input인 컴포넌트의 예시입니다.

var strRet;

//inputbox의 초기 border-color style 값 반환받기.
strRet = ibx_input.getInitStyle("border-color");

5주요 API
