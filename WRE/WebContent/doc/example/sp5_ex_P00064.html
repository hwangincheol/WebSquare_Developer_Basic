[AutoComplete] 검색어 강조하기

1개요

사용자가 입력한 검색어를 강조하는 기능인 useKeywordHighlight 속성 예제입니다.

P00064

2구현된 기능

  • 검색어 강조하지 않기(기본 설정)

  • 검색어 강조하기

3예제 테스트 방법

검색어를 "b" 또는 "B" 로 입력하여 출력된 목록의 검색어 강조를 비교합니다.

그림 1.브라우저(Chrome) 실행 예시 - 검색어 강조 안함

그림 2.브라우저(Chrome) 실행 예시 - 검색어 강조함

4구현 예시

4.1검색어 강조하기

컴포넌트의 useKeywordHighlight 속성을 true로 정의합니다. 이 속성을 적용하지 않은 경우 기본값은 false(검색어 강조하지 않음)입니다.

그림 3.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

<!-- autoComplete의 소스 본문 예시 -->
<w2:autoComplete useKeywordHighlight="true" >
<!-- 중략 -->
</w2:autoComplete>

5주요 API

  • useKeywordHighlight
