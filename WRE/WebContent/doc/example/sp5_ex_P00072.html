[Calendar] 달력의 요일을 이미지 또는 텍스트로 표시하기

1개요

컴포넌트의 요일 영역을 브라우저에 출력하는 방법의 설정 예제입니다.  이미지로 출력 또는 문자열로 출력할 수 있습니다.
속성 'displayDayType'를 설정하여 구현할 수 있습니다.

P00072

2구현된 기능

  • 요일 영역을 이미지(기본 설정값)로 출력

  • 요일 영역을 문자열로 출력

3예제 테스트 방법

설정별로 구성된 컴포넌트의 요일 영역의 출력 방법을 확인합니다.

예제 프로젝트의 경우 이미지와 문자열의 차이가 확연하게 드러나지 않습니다.

브라우저의 개발자 도구를 통해 출력된 요소(elements)를 명확하게 비교할 수 있습니다.

3.1요일 영역을 이미지(기본 설정값)로 출력

그림 1.브라우저(Chrome) 실행 예시 - 요일 영역을 이미지(기본 설정값)로 출력된 상태

3.2요일 영역을 문자열로 출력

그림 2.브라우저(Chrome) 실행 예시 - 요일 영역을 문자열로 출력

4구현 예시

4.1요일 영역을 문자열로 출력(표현)하기

컴포넌트의 속성 displayDayType을 text로 정의합니다. 이 속성을 적용하지 않은 경우 기본값은 image입니다.

그림 3.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

<!-- calendar의 소스 본문 예시 -->
<w2:calendar displayDayType="text">
</w2:calendar>

5주요 API

displayDayType
