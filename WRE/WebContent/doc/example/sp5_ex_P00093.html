[GridView] 특정 컬럼의 선택 불가 설정하기

1개요

GridView의 특정 컬럼을 선택하지 못하도록 설정하는 예제입니다.

P00093

2구현된 기능

  • [기본 설정] 모든 컬럼의 선택이 가능한 상태

  • 특정 컬럼의 선택이 불가한 상태- 속성으로 설정

  • 특정 컬럼의 선택이 불가한 상태- 스크립트로 설정

3예제 테스트 방법

설정별 영역의 GridView의 "도서명" 컬럼의 Cell을 클릭하여 선택(포커스) 여부를 비교합니다.

3.1모든 컬럼의 선택이 가능한 상태

GridView 셀을 클릭합니다.

"[기본 설정] 모든 컬럼의 선택이 가능한 상태" 영역의 GridView의 1번째 셀을 클릭합니다.

셀이 선택됨을 확인합니다.

[브라우저(Chrome) 실행 예시 - 셀 선택 예시]

3.2"도서명" 컬럼의 선택 불가 설정 - 속성

STEP1. GridView의 컬럼의 "도서명"의 셀을 클릭합니다.

"도서명" 컬럼의 선택 불가 설정 - 속성 영역의 GridView의 1번째 셀을 클릭합니다.

셀이 선택되지 않는 것을 확인합니다.

[브라우저(Chrome) 실행 예시 - 셀 선택 예시]

STEP2. GridView의 컬럼 "정가"의 셀을 클릭합니다.

셀이 선택되는 것을 확인합니다.

[브라우저(Chrome) 실행 예시 - 셀 선택 예시]

3.3"도서명" 컬럼의 선택 불가 설정 - 스크립트

STEP1. GridView의 컬럼의 "도서명"의 셀을 클릭합니다.

"도서명" 컬럼의 선택 불가 설정 - 스크립트 영역의 GridView의 1번째 셀을 클릭합니다.

셀이 선택되는 것을 확인합니다.

[브라우저(Chrome) 실행 예시 - 셀 선택 예시]

STEP2. 버튼 "도서명" 컬럼에 선택 불가 설정하기를 클릭합니다.

-

STEP3. GridView의 컬럼의 "도서명"의 셀을 클릭합니다.

셀이 선택되지 않는 것을 확인합니다.

[브라우저(Chrome) 실행 예시 - 셀 선택 예시]

4구현 예시

DataList 생성 및 연결은 생략되었습니다.

4.1GridView의 컬럼 선택 불가 설정하기 - 속성

GridView의 컬럼 "도서명"의 속성을 정의합니다.

  • [필수] blockSelect="true" //[default: false, true] 해당 컬럼의 선택을 막을지에 대한 여부를 정의입니다.

STEP1. 디자인탭에서 GridView의 컬럼 "도서명"를 선택합니다.

[웹스퀘어5 SP5 스튜디오의 Design 탭 예시]

STEP2. GridView 컬럼의 속성을 설정합니다.

[웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시]

<!-- gridView 의 소스 본문 예시 -->
<w2:gridView dataList="data:dlt_books">
    <!-- 중략 -->
    <w2:column blockSelect="true" id="book_name">
    </w2:column>
    <!-- 중략 -->
</w2:gridView>

4.2GridView의 컬럼 선택 불가 설정하기 - 스크립트로 설정하기

원하는 시점에 setBlockSelect method를 이용하여 스크립트에서 설정할 수 있습니다.

//컬럼 "도서명"에 선택 불가 설정하기
grd_exam3.setBlockSelect("book_name", true);

5주요 API

  • blockSelect

  • setBlockSelect( colIndex , flag )
