[GridView] 마우스 DragDrop 기능을 통한 GridView 내 행의 이동 금지 설정하기

1개요

GridView의 Drag&Drop 기능의 옵션 중 하나인 "dataDragDropOther" 속성 예제입니다.
이 기능은 Drag&Drop 기능을 통한 GridView 내부에서의 행 이동을 금지하는 기능을 제공합니다.

P00102

2구현된 기능

  • DragDrop 설정 + GridVIew 내 행 이동 금지

3예제 테스트 방법

3.1GridView내 Drag&Drop으로 행 이동 금지

GridView 내의 행을 Drag하여 GridView 내에서 Drop할 경우 이동되지 않음을 확인합니다.

STEP1. GridView의 1번째 행 컬럼 "도서명"을 선택합니다.

[브라우저(Chrome) 실행 예시]

STEP2. GridView의 1번째 행을 Drag하여 3번째 행에 Drop합니다.

[브라우저(Chrome) 실행 예시]

STEP3. 실행 결과를 확인합니다.

Drag한 행이 이동되지 않음을 확인합니다.

3.2타 GridView에 Drop으로 행 이동

STEP1. GridView의 1번째 행 컬럼 "도서명"을 선택합니다.

4구현 예시

GridView와 연결된 DataList 생성 및 연결 방법은 생략되었습니다.

4.1GridView내 Drag&Drop으로 행 이동 금지 설정하기

GridView의 속성을 정의합니다.

  • [필수] dataDragDrop="true" //[default:false, true] 동일 gridView 또는 서로 다른 gridView 간의 데이터 드래그-앤-드롭을 허용

  • [필수] dataDragDropOther="true" //[default:false, true] 서로 다른 gridView 간의 데이터 드래그-앤-드롭만 허용할지 여부

그림 1.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

[소스 코드 예시]

<!-- gridView 의 소스 본문 예시 -->
<w2:gridView dataDragDrop="true" dataDragDropOther="true" dataList="data:dlt_books_1">
    <!-- 중략 -->
</w2:gridView>

5주요 API

  • dataDragDrop

  • dataDragDropOther
