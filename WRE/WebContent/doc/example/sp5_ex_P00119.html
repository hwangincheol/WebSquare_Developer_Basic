[GridView] 병합된 헤더 컬럼에 정렬 기능을 사용할 바디 컬럼 지정하기

1개요

GridView의 헤더 컬럼이 병합되었을 때 정렬 기능을 적용할 바디 컬럼을 지정하는 예제입니다.

P00119

2구현된 기능

  • [기본 동작] 병합된 헤더 컬럼의 정렬 기능

  • 헤더 컬럼에 정렬 기능을 적용할 바디 컬럼 지정하기

3예제 테스트 방법

3.1[기본 동작] 병합된 헤더 컬럼의 정렬 기능

STPE1. 헤더 컬럼의 정렬 기능 아이콘을 클릭합니다.

영역 [기본 설정]의 GridView의 헤더 컬럼의 정렬 기능 아이콘을 클릭합니다.

[브라우저(Chrome) 실행 예시]

STEP2. 실행된 결과를 확인합니다.

정렬 기능이 병합된 헤더 컬럼의 첫번째 바디 컬럼(도서명)에 적용되어 동작합니다.

첫번째 바디 컬럼이 오름 차순으로 정렬됩니다.

[브라우저(Chrome) 실행 예시]

3.2병합된 헤더 컬럼의 정렬 기능을 두번째 바디 컬럼에 지정하기

STPE1. 헤더 컬럼의 정렬 기능 아이콘을 클릭합니다.

영역 [두번째 바디 컬럼(정가)이 정렬]의 GridView의 헤더 컬럼의 정렬 기능 아이콘을 클릭합니다.

[브라우저(Chrome) 실행 예시]

STEP2. 실행된 결과를 확인합니다.

정렬 기능이 병합된 헤더 컬럼의 두번째 바디 컬럼(정가)에 적용되어 동작합니다.

첫번째 두번째 컬럼이 오름 차순으로 정렬됩니다.

[브라우저(Chrome) 실행 예시]

4구현 예시

GridView와 연결된 DataList 생성 및 연결 방법은 생략되었습니다.

4.1병합된 헤더 컬럼의 정렬 기능을 특정 바디 컬럼에 지정하기

STEP1. GridView의 속성을 정의합니다.

  • [필수] sortable="true" //gridView의 헤더 클릭을 통한 데이터 정렬 지원 여부

  • [선택] showSortableImage="true" //정렬 가능한 컬럼의 헤더에 정렬 이미지를 출력.

  • [선택] sortEvent="onclick" //[default: ondblclick, onclick] 정렬을 수행할 이벤트를 정의.

그림 1.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

STEP2. GridView의 헤더 컬럼의 속성을 정의합니다.

  • [필수] orderByColumn="바디 컬럼 ID" //정렬 기능을 적용할 바디 컬럼의 ID

    ex) orderByColumn="price"

그림 2.웹스퀘어5 SP5 스튜디오의 Design 탭 예시 - 헤더 컬럼 선택

그림 3.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

[소스 코드 예시]

<!-- gridView 의 소스 본문 예시 -->
<w2:gridView sortable="true" sortEvent="onclick" showSortableImage="true" dataList="data:dlt_books_2">
    <!-- 중략 -->
    <w2:header id="header1" style="">
        <w2:row id="row1" style="">
            <w2:column orderByColumn="price" value="도서 정보" colSpan="3" id="column3"></w2:column>
            <!-- 중략 -->
        </w2:row>
    </w2:header>
    <w2:gBody id="gBody1" style="">
        <w2:row id="row2" style="">
            <!-- 중략 -->
            <w2:column id="price" dataType="number" displayFormat="#,###"></w2:column>
            <!-- 중략 -->
        </w2:row>
    </w2:gBody>
</w2:gridView>
