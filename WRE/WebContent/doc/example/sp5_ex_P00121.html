[Group] Event - onmouseleave (마우스 포인터가 영역 밖으로 나갈 때 발생)

1개요

Group의 이벤트 onmouseleave 예제입니다.
이벤트 onmouseleave는 마우스 포인터가 컴포넌트의 영역 밖으로 나갔을 때 발생합니다.

P00121

2구현된 기능

  • 이벤트 onmouseleave 발생 시 로그 출력하기

3예제 테스트 방법

3.1이벤트 onmouseleave 발생 시 로그 출력하기

STEP1. 마우스를 Group 영역으로 이동합니다.

[브라우저(Chrome) 실행 예시 - Group 영역 안에 마우스 포인터가 위치]

STEP2. 마우스를 Group 영역 밖으로 이동합니다.

[브라우저(Chrome) 실행 예시 - Group 영역 밖으로 마우스 포인터가 위치]

STEP3. 실행 결과를 확인합니다.

마우스 포인터가 밖으로 이동되면 영역 [로그 확인]에 이벤트 정보가 출력됩니다.

브라우저의 개발자 도구의 콘솔(console)탭을 통해 자세한 이벤트 정보를 확인 할 수 있습니다.

[브라우저(Chrome) 실행 예시 - 로그 확인]

4.1이벤트 onmouseleave 발생 시 로그 출력하기

STEP1. Group의 이벤트 onmouseleave 핸들러를 지정합니다.

예제 파일에서는 핸들러로 사용할 함수명을 scwin.grp_exam1_onmouseleave로 정의하였습니다.

그림 1.웹스퀘어5 SP5 스튜디오의 Property View(속성창)의 이벤트 탭 예시

[소스 코드 예시]

<!-- group 의 소스 본문 예시 -->
<xf:group ev:onmouseleave="scwin.grp_exam1_onmouseleave" id="grp_exam1">
</xf:group>
