[Input] 최대 입력 길이 제한하기 - maxLength, maxByteLength

1개요

Input의 입력값의 길이를 제한하는 예제입니다.
속성 maxLength, maxByteLength를 비교할 수 있습니다.
속성별 특징은 다음과 같습니다.

  • maxLength : 글자 수를 기준으로 제한합니다.

  • maxByteLength : 입력값을 byte로 계산하여 제한합니다.

    client 설정파일에 적용된 byteCheckEncoding의 설정값을 기준으로 아래와 같이 적용됩니다.

    - utf-8 : 한글 1글자가 3byte

    - euc-kr : 한글 1글자가 2byte

P00127

2구현된 기능

  • 기본 설정

  • 입력값을 최대 4글자로 제한 - maxLength : 4

  • 입력값을 최대 4byte로 제한 - maxByteLength : 4

3예제 테스트 방법

3.1입력값을 최대 4글자로 제한

STEP1. 영역 [입력값을 최대 4글자로 제한]의 Input에 값 "일이삼사오"을 입력합니다.

[브라우저(Chrome) 실행 예시]

STEP2. 컴포넌트의 외부 영역을 클릭하여 커서를 제거합니다.

[브라우저(Chrome) 실행 예시]

STEP3. 실행 결과를 확인합니다.

Input의 입력 영역을 클릭하여 입력값을 확인합니다.

입력한 값이 "일이삼사"으로 변경됨을 확인합니다.

[브라우저(Chrome) 실행 예시]

3.2입력값을 최대 4byte로 제한

STEP1. 영역 [입력값을 최대 4byte로 제한]의 Input에 값 "일이"을 입력합니다.

[브라우저(Chrome) 실행 예시]

STEP2. 실행 결과를 확인합니다.

입력한 값이 "일"로 제한됨을 확인합니다.

[브라우저(Chrome) 실행 예시]

4구현 예시

4.1입력값을 최대 4글자로 제한

Input의 속성을 정의합니다.

  • [필수] maxlength="4" //입력 가능한 최대길이를 지정합니다.

그림 1.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

[소스 코드 예시]

<!-- input 의 소스 본문 예시 -->
<xf:input maxlength="4"> </xf:input>

4.2입력값을 최대 4byte로 제한

Input의 속성을 정의합니다.

  • [필수] maxBytelength="4" //입력 가능한 최대 Byte길이를 지정합니다.

그림 2.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

[소스 코드 예시]

<!-- input 의 소스 본문 예시 -->
<xf:input maxBytelength="4"> </xf:input>
