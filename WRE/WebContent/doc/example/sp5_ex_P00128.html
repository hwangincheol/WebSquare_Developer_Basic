[Input] Event - oneditfull (입력값이 maxByteLength 설정값과 같을 때 발생)

1개요

Input의 이벤트 oneditfull 예제입니다.
이벤트 oneditfull는 입력값이 속성 maxByteLength의 설정값과 동일할 때 발생합니다.

이 기능은 입력값이 영문, 숫자이고, 속성 maxByteLength이 설정된 경우만 동작합니다.

P00128

2구현된 기능

  • 이벤트 oneditfull 발생 시 로그 출력하기

3예제 테스트 방법

3.1이벤트 oneditfull 발생 시 로그 출력하기

STEP1. 영역 [이벤트 oneditfull 설정]의 Input에 값 "1234"을 입력합니다.

[브라우저(Chrome) 실행 예시]

STEP2. 실행 결과를 확인합니다.

입력값이 "1234"가 되면 이벤트 oneditfull가 발생합니다.

oneditfull 핸들러에 작성된 스크립트가 실행됩니다.

영역 [로그 확인]에 컴포넌트 ID와 value가 출력됩니다.

[브라우저(Chrome) 실행 예시]
