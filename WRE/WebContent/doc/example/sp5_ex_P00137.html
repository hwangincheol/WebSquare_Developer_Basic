[InputCalendar] 오늘 날짜 할당하기

1개요

InputCalendar에 오늘 날짜를 할당(출력)하는 예제입니다.

P00137

2구현된 기능

  • 오늘 날짜 출력하기 - 속성

  • 오늘 날짜 출력하기 - 스크립트

3예제 테스트 방법

3.1오늘 날짜 출력하기 - 속성

실행 결과를 확인합니다.

영역 [오늘 날짜 출력하기 - 속성]의 InputCalendar에 출력된 value를 확인합니다.

예제 파일이 실행된 날짜가 출력되는 것을 확인할 수 있습니다.

[브라우저(Chrome) 실행 예시]

3.2오늘 날짜 출력하기 - 스크립트

STEP1. 스크립트 실행하기

영역 [오늘 날짜 출력하기 - 스크립트]의 버튼 오늘 날짜 할당을 클릭합니다.

[브라우저(Chrome) 실행 예시]

STEP2. 실행 결과를 확인합니다.

InputCalendar에 오늘 날짜(예제 파일이 실행된 날짜)가 value로 할당된 것을 확인합니다.

[브라우저(Chrome) 실행 예시]

4구현 예시

4.1오늘 날짜 출력하기 - 속성

속성을 지정합니다.

  • [필수] setCurrentDate="true" //날짜 input 현재일을 출력

그림 1.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

[소스 코드 예시]

<!-- inputCalendar 의 소스 본문 예시 -->
<w2:inputCalendar setCurrentDate="true" calendarValueType="yearMonthDate" id="ica_exam_1">
</w2:inputCalendar>

4.2오늘 날짜 출력하기 - 스크립트

스크립트를 작성합니다.

원하는 시점에 스크립트를 작성합니다.

이 예제는 스크립트 "scwin.btn_ex1_onclick"에 작성되었습니다.

[소스 코드 예시]

//예제 파일의 스크립트 "scwin.btn_ex1_onclick"를 참고하세요.

var strDate;

//오늘 날짜 반환 받기 - InputCalenar에 설정된 ioFormat과 동일해야 합니다.
strDate = $p.getCurrentServerDate("yyyyMMdd");

//InputCalenar [ica_exam_2]에 value 할당하기
ica_exam_2.setValue(strDate);

5주요 API

  • setCurrentDate

  • ioFormat

  • setValue( value )

  • $p.getCurrentServerDate( pattern )
