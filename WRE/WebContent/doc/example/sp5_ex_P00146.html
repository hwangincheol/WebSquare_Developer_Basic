[WindowContainer] 윈도우를 닫았을 때 남아 있는 윈도우를 자동 정렬 하기

1개요

윈도우를 닫으면 남은 윈도우가 자동 정렬되는 기능 예제입니다.
속성 autoArrange 지정을 통해 사용할 수 있습니다.
이 기능은 정렬 기능이 설정된 경우 동작합니다.

P00146

2구현된 기능

  • 윈도우 자동 정렬 기능 설정

  • (기본) 윈도우 자동 정렬 기능 미설정

3예제 테스트 방법

3.1윈도우 자동 정렬 기능 설정

STEP1. 윈도우를 닫습니다.

영역 [윈도우 자동 정렬 기능 설정]의 WindowContainer의 첫번째 윈도우를 닫습니다.

[브라우저(Chrome) 실행 예시]

STEP2. 실행 결과를 확인합니다.

남은 윈도우가 재 정렬됩니다.

[브라우저(Chrome) 실행 예시]

3.2(기본) 윈도우 자동 정렬 기능 미설정

STEP1. 윈도우를 닫습니다.

영역 [기본 설정]의 WindowContainer의 첫번째 윈도우를 닫습니다.

[브라우저(Chrome) 실행 예시]

STEP2. 실행 결과를 확인합니다.

남은 윈도우의 위치가 동일하게 유지됩니다.

[브라우저(Chrome) 실행 예시]

4구현 예시

4.1윈도우 자동 정렬 기능 설정하기

WindowContainer의 속성을 정의합니다.

  • [필수] autoArrange="true" //윈도우 재정렬 지정

그림 1.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

[소스 코드 예시]

<!-- windowContainer 의 소스 본문 예시 -->
<w2:windowContainer autoArrange="true" id="wdc_exam2">
    <!-- 중략 -->
</w2:windowContainer>

5주요 API

  • autoArrange
