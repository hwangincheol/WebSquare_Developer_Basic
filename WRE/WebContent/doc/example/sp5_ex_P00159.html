[TabControl] 탭을 표시할 위치를 지정하기

1개요

TabControl의 탭을 표시할 위치를 지정하는 속성 tabPosition 예제입니다.
속성 tabPosition에 지정할 수 있는 값은 "top", "left", "right", "bottom" 입니다.
속성 tabScroll이 "true"로 지정된 경우 "top", "bottom"만 지원되며, "left", "right"로 지정된 경우 "top"으로 동작합니다.

2구현된 기능

  • 탭 위치 : right

  • 탭 위치 : bottom

  • 탭 위치 : left

4구현 예시

4.1탭 위치 지정하기

속성을 정의합니다.

지정할 수 있는 값은 "top", "left", "right", "bottom" 입니다.

속성 tabScroll이 "true"로 지정된 경우 "top", "bottom"만 지원되며, "left", "right"로 지정된 경우 "top"으로 동작합니다.

  • [필수] tabPosition="right" //탭이 오른쪽에 위치됩니다.

    (지정값 설명)

    "top" : 위에 위치.

    "left" : 왼쪽에 위치.

    "right" : 오른쪽에 위치.

    "bottom" : 하단에 위치.

<!-- tabControl 소스 본문 예시 -->

    <!-- 중략 -->
