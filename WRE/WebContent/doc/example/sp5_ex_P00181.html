[GridView] 행 번호의 시작 값 지정하기

1개요

GridView의 행 번호의 시작 값을 지정하는 예제입니다.
이 기능은 함수 setStartRowNumber을 호출하여 사용할 수 있습니다.

P00181

2구현된 기능

  • 행 번호의 시작 값 지정하기

3예제 테스트 방법

3.1행 번호의 시작 값 지정하기

STEP1. 초기 행 번호를 확인합니다.

영역 [행 번호의 시작 값 설정하기]의 GridView의 행 번호 시작 값을 확인합니다.

[브라우저(Chrome) 실행 예시 - 초기 값]

STEP2. 행 번호 시작 값을 변경합니다.

버튼 시작 값를 11로 지정하기을 클릭합니다.

STEP3. 실행 결과를 확인합니다.

GridView의 행 번호 시작 값이 11로 변경됩니다.

[브라우저(Chrome) 실행 예시]

4구현 예시

4.1행 번호의 시작 값 지정하기

STEP1. GridView의 속성을 정의합니다.

  • [필수] rowNumVisible="true" //[default:false, true] 행 번호 표시 여부

그림 1.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

[소스 코드 예시]

<!-- gridView 의 소스 본문 예시 -->
<w2:gridView rowNumVisible="true" dataList="data:dlt_books_1" style="height: 100px;">
    <!-- 중략 -->
</w2:gridView>
STEP2. 원하는 시점에 스크립트를 작성합니다.

GridView의 함수 setStartRowNumber을 사용합니다.

[소스 코드 예시]

//예제 파일의 스크립트 "scwin.btn_ex1_onclick"를 참고하세요.

//GridView [grd_exam1]의 행 번호 시작 값을 11로 지정합니다.
grd_exam1.setStartRowNumber(10);

5주요 API

  • rowNumVisible

  • setStartRowNumber( rowIndex )
