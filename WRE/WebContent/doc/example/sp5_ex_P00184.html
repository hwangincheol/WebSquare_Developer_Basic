[GridView] 컬럼의 너비를 그리드 너비에 맞춰 자동 조절 지정하기 및 해제하기 - 스크립트

1개요

GridView의 속성 autoFit을 스크립트로 지정하는 예제입니다.
GridView의 함수 setAutoFit, clearAutoFit으로 제어할 수 있습니다.
- 함수 setAutoFit : autoFit 속성을 지정합니다.
- 함수 clearAutoFit : autoFit 속성을 해제합니다.

P00184

2구현된 기능

  • 속성 autoFit 설정 값을 allColumn으로 지정하기

  • 속성 autoFit 설정 값을 lastColumn으로 지정하기

  • 속성 autoFit 설정 해제하기

3예제 테스트 방법

3.1속성 autoFit 설정 값을 allColumn으로 지정하기

STEP1. 초기 상태 확인하기

영역 [속성 autoFit 설정 값을 allColumn으로 지정하기]의 GridView를 확인합니다.

우측에 여백이 있습니다.

[브라우저(Chrome) 실행 예시]

STEP2. autoFit 설정 값을 변경합니다.

버튼 autoFit의 설정 값을 'allColumn'으로 지정하기를 클릭합니다.

STEP3. 실행 결과를 확인합니다.

여백이 각 컬럼에 균등하게 분할됩니다.

[브라우저(Chrome) 실행 예시]

3.2속성 autoFit 설정 값을 lastColumn으로 지정하기

STEP1. 초기 상태 확인하기

영역 [속성 autoFit 설정 값을 lastColumn으로 지정하기]의 GridView를 확인합니다.

우측에 여백이 있습니다.

[브라우저(Chrome) 실행 예시]

STEP2. autoFit 설정 값을 변경합니다.

버튼 autoFit의 설정 값을 'lastColumn'으로 지정하기를 클릭합니다.

STEP3. 실행 결과를 확인합니다.

여백이 마지막 컬럼에 할당됩니다.

[브라우저(Chrome) 실행 예시]

3.3속성 autoFit 설정 해제하기

STEP1. 초기 상태 확인하기

영역 [속성 autoFit 설정 해제하기]의 GridView를 확인합니다.

속성 autoFit의 값이 'lastColumn'으로 지정되어, 마지막 컬럼에 여백이 할당된 상태입니다.

[브라우저(Chrome) 실행 예시]

STEP2. autoFit 설정을 해제합니다.

버튼 속성 autoFit 설정 해제하기를 클릭합니다.

STEP3. 실행 결과를 확인합니다.

autoFit 기능이 해제되어 우측에 여백이 생성됩니다.

[브라우저(Chrome) 실행 예시]

4구현 예시

4.1속성 autoFit 설정 값을 allColumn으로 지정하기

원하는 시점에 스크립트를 작성합니다.

GridView의 함수 setAutoFit을 사용합니다.

[소스 코드 예시]

//예제 파일의 스크립트 "scwin.btn_ex1_onclick"를 참고하세요.

//autoFit 설정 옵션
var jsnAutoFitOptions = {
    type : "allColumn"  //autoFit 타입
    //, minWidth : "300" //설정 값은 px 단위이며, gridView의 width가 설정 값보다 작아질 경우에 각 컬럼의 width의 합이 설정 값로 고정되어 계산됩니다. GridView의 속성 autoFitMinWidth과는 다른 기능입니다.
};

//GridView [grd_exam1]에 autoFit 속성을 지정합니다.
grd_exam1.setAutoFit(jsnAutoFitOptions);

4.2속성 autoFit 설정 값을 lastColumn으로 지정하기

원하는 시점에 스크립트를 작성합니다.

GridView의 함수 setAutoFit을 사용합니다.

[소스 코드 예시]

//예제 파일의 스크립트 "scwin.btn_ex2_onclick"를 참고하세요.

//autoFit 설정 옵션
var jsnAutoFitOptions;

jsnAutoFitOptions = {
    type : "lastColumn"  //autoFit 타입
};

//GridView [grd_exam2]에 autoFit 속성을 지정합니다.
grd_exam2.setAutoFit(jsnAutoFitOptions);

4.3속성 autoFit 설정 해제하기

원하는 시점에 스크립트를 작성합니다.

GridView의 함수 clearAutoFit을 사용합니다.

[소스 코드 예시]

//예제 파일의 스크립트 "scwin.btn_ex3_onclick"를 참고하세요.

//GridView [grd_exam3]에 autoFit 속성을 해제합니다.
grd_exam3.clearAutoFit();

5유의 사항

함수 setAutoFit은 현재 그리드의 상태(여백, 컬럼 폭의 합 등)를 기준으로 계산됩니다.

최초 그려진 GridView를 기준으로 재 설정하고자 하는 경우는 먼저 함수 clearAutoFit으로 설정을 해제해야 합니다.

6주요 API

  • setAutoFit( options )

  • clearAutoFit( )
