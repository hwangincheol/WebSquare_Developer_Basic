[GridView] 셀의 배경색 지정하기

1개요

GridView의 특정 셀의 배경색을 지정하는 예제입니다.
이 기능은 함수 setCellBackgroundColor를 호출하여 사용할 수 있습니다.

P00185

2구현된 기능

  • 셀의 배경색 지정하기

3예제 테스트 방법

3.1셀의 배경색 지정하기

STEP1. 초기 상태 확인하기

영역 [GridView 셀의 배경색 지정하기]의 GridView를 확인합니다.

GridView의 셀의 기본 배경색이 지정되어 있지 않습니다.

[브라우저(Chrome) 실행 예시]

STEP2. 셀의 배경색을 변경합니다.

버튼 컬럼 '정가'의 2번째 행의 배경색을 'peru'로 지정하기을 클릭합니다.

STEP3. 실행 결과를 확인합니다.

컬럼 '정가'의 2번째 행의 셀의 배경색이 'peru'로 변경됩니다.

[브라우저(Chrome) 실행 예시]

STEP4. 셀의 배경색을 변경합니다.

버튼 컬럼 '도서명'의 1번째 행의 배경색을 '#9ACD32'로 지정하기을 클릭합니다.

STEP5. 실행 결과를 확인합니다.

컬럼 '도서명'의 1번째 행의 배경색이 '#9ACD32'로 변경됩니다.

[브라우저(Chrome) 실행 예시]

4구현 예시

4.1셀의 배경색 지정하기

원하는 시점에 스크립트를 작성합니다.

GridView의 함수 setCellBackgroundColor을 사용합니다.

[소스 코드 예시]

//예제 파일의 스크립트 "scwin.btn_ex1_onclick" 또는 "scwin.btn_ex2_onclick"를 참고하세요.

//GridView 'grd_exam1'의 컬럼 '정가'의 2번째 행의 배경색을 'peru'로 지정하기
grd_exam1.setCellBackgroundColor(1, "price", "peru");

//GridView 'grd_exam1'의 컬럼 '도서명'의 1번째 행의 배경색을 '#9ACD32'로 지정하기
grd_exam1.setCellBackgroundColor(0, "book_name", "#9ACD32");

5주요 API

  • setCellBackgroundColor( rowIndex , colIndex , color )
