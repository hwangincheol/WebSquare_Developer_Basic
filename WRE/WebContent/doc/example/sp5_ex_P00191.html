[GridView] GridView 컬럼의 속성 dataType, 속성 displayFormat을 스크립트로 지정하기

1개요

GridView 컬럼의 속성 dataType과 속성 displayFormat의 값을 지정하고 속성 dataType의 값을 반환받는 예제입니다.
지정 가능한 dataType 값은 아래와 같습니다.
- text, number, float
이 예제에서 사용한 함수는 다음과 같습니다.
- getDataType : 컬럼의 속성 dataType의 값을 반환.
- setDataType : 컬럼의 속성 dataType의 값을 지정.
- setColumnDisplayFormat : 컬럼의 속성 displayFormat의 값을 지정.

P00191

2구현된 기능

  • 컬럼의 속성 dataType의 설정 값 반환받기

  • 컬럼의 속성 dataType을 'number'로 지정하고 속성 dispalyFormat을 '#,###'으로 스크립트로 지정하기

  • 컬럼의 속성 dataType을 'float'로 지정하고 속성 dispalyFormat을 '#,###.00'으로 스크립트로 지정하기

3예제 테스트 방법

이 예제는 예제 화면의 영역 [로그 확인]의 textarea와 브라우저 개발자 도구의 콘솔에 반환 받은 컴포넌트 정보가 출력됩니다.

4구현 예시

4.1컬럼의 속성 dataType의 설정 값 반환받기

원하는 시점에 스크립트를 작성합니다.

GridView의 함수 getDataType을 사용합니다.

[소스 코드 예시]

//예제 파일의 스크립트 "scwin.btn_ex1_1_onclick"을 참고하세요.

var returnValue;

//GridView 'grd_exam1'의 컬럼 '정가'의 속성 dataType의 값을 반환 받습니다.
returnValue = grd_exam1.getDataType("price");

4.2컬럼의 속성 dataType의 설정 값 스크립트로 지정하기

원하는 시점에 스크립트를 작성합니다.

GridView의 함수 setDataType을 사용합니다.

지정 가능한 dataType 값은 아래와 같습니다.

- text, number, float

[소스 코드 예시]

//예제 파일의 스크립트 "scwin.btn_ex1_2_onclick" 또는 "scwin.btn_ex1_3_onclick"을 참고하세요.

//GridView 'grd_exam1'의 컬럼 '정가'의 속성 dataType의 값을 'number'로 지정합니다.
grd_exam1.setDataType("price", "number");

//GridView 'grd_exam1'의 컬럼 '정가'의 속성 dataType을 'float'로 지정합니다.
grd_exam1.setDataType("price", "float");

//GridView 'grd_exam1'의 컬럼 '정가'의 속성 dataType을 'text'로 지정합니다.
grd_exam1.setDataType("price", "text");

4.3컬럼의 속성 displayFormat의 설정 값 스크립트로 지정하기

원하는 시점에 스크립트를 작성합니다.

GridView의 함수 setColumnDisplayFormat을 사용합니다.

[소스 코드 예시]

//예제 파일의 스크립트 "scwin.btn_ex1_2_onclick" 또는 "scwin.btn_ex1_3_onclick"을 참고하세요.

//GridView 'grd_exam1'의 컬럼 '정가'의 속성 displayFormat의 값을 '#,###'으로 지정합니다.
grd_exam1.setColumnDisplayFormat("price", "#,###");

//GridView 'grd_exam1'의 컬럼 '정가'의 속성 dispalyFormat을 '#,###.00'으로 지정합니다.
grd_exam1.setColumnDisplayFormat("price", "#,###.00");

5주요 API

  • getDataType( colID )

  • setDataType( colIndex , dataType )

  • setColumnDisplayFormat( colIndex , displayFormat )
