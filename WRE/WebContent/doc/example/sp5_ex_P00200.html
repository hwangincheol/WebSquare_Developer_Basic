[GridView] 행 고정하기

1개요

GridView의 행 고정 기능 예제입니다.
이 기능은 아래의 속성과 함수로 사용할 수 있습니다.
- fixedRow : (속성) 행 고정
- setFixedRow : (함수) 행 고정

P00200

2구현된 기능

  • 행 고정 적용 - 속성

  • 행 고정 적용 - 스크립트

3예제 테스트 방법

3.1행 고정 적용 - 속성

STEP1. 실행 결과를 확인합니다.

영역 [행 고정 적용 - 속성]의 GridView를 확인합니다.

상단에서 2번째 행이 고정되었습니다.

[브라우저(Chrome) 실행 예시]

3.2행 고정 적용 - 스크립트

STEP1. 초기 상태를 확인합니다.

영역 [행 고정 적용 - 스크립트]의 GridView를 확인합니다.

행 고정이 적용되지 않았습니다.

[브라우저(Chrome) 실행 예시]

STEP2. 행을 고정합니다.

버튼 상단 1번째 행까지 고정하기를 클릭합니다.

상단에서 1번째 행이 고정됩니다.

[브라우저(Chrome) 실행 예시]

STEP3. 고정된 행을 해제합니다.

버튼 행 고정 모두 해제하기를 클릭합니다.

고정된 행이 모두 해제됩니다.

[브라우저(Chrome) 실행 예시]

4구현 예시

4.1행 고정 적용 - 속성

GridView의 속성을 정의합니다.

  • [필수] fixedRow="설정 값" //상단에서 고정할 행의 위치

    예시) fixedRow="1" //상단에서 1번째 행을 고정

그림 1.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

[소스 코드 예시]

<!-- gridView 의 소스 본문 예시 -->
<w2:gridView fixedRow="1" dataList="data:dlt_books" style="height: 100px;">
    <!-- 중략 -->
</w2:gridView>

4.2행 고정 적용 - 스크립트

원하는 시점에 스크립트를 작성합니다.

GridView의 함수 setFixedRow 을 사용합니다.

[소스 코드 예시]

//예제 파일의 스크립트 "scwin.btn_ex1_1_onclick", "scwin.btn_ex1_2_onclick"을 참고하세요.

//GridView 'grd_exam2'의 상단 1번째 행까지 고정하기
grd_exam2.setFixedRow(1);

//GridView 'grd_exam2'의 행 고정 해제하기
grd_exam2.setFixedRow(0);

5주요 API

  • fixedRow

  • setFixedRow( fixedRowNum )
