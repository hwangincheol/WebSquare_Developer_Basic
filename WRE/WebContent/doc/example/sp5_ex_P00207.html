[GridView] 보여질 행의 개수 지정하기

1개요

GridView에 보여질 행의 개수를 지정하는 예제입니다.
이 기능은 아래의 속성과 함수로 사용할 수 있습니다.
- visibleRowNum : (속성) 화면에 표현할 행의 개수.
- setVisibleRowNum : (함수) 속성 visibleRowNum을 스크립트로 지정.

P00207

2구현된 기능

  • 보여질 행의 수 지정하기

  • 속성 visibleRowNum을 미지정

3예제 테스트 방법

3.1보여질 행의 수 지정하기

STEP1. 초기 상태 확인하기

영역 [보여질 행의 수 지정하기]의 GridView를 확인합니다.

GridView에 보여지는 행의 수는 3개입니다.

[브라우저(Chrome) 실행 예시]

STEP2. 보여지는 행의 수를 2개로 변경합니다.

버튼 보여지는 행을 2개로 지정하기를 클릭합니다.

STEP3. 실행 결과를 확인합니다.

GridView에 보여지는 행이 2개로 변경됩니다.

[브라우저(Chrome) 실행 예시]

STEP4. 보여지는 행의 수를 전체로 변경합니다.

버튼 보여지는 행을 전체로 지정하기를 클릭합니다.

STEP5. 실행 결과를 확인합니다.

GridView에 보여지는 행이 전체로 변경됩니다.

[브라우저(Chrome) 실행 예시]

3.2속성 visibleRowNum을 미지정

STEP1. 실행된 결과를 확인합니다.

영역 [속성 visibleRowNum을 미지정]의 GridView를 확인합니다.

GridView에 지정된 높이 100px에 출력될 수 있는 최대의 행이 표시됩니다.

1개의 행이 표시되고 남은 여백은 공백으로 표시됩니다.

[브라우저(Chrome) 실행 예시]

4구현 예시

4.1보여질 행의 수 지정하기 - 속성

GridView의 속성을 정의합니다.

  • [필수] visibleRowNum="설정값" //화면에 보여질 행의 수

    예시1) visibleRowNum="3" //화면에 보여질 행의 수를 3개로 지정

그림 1.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

[소스 코드 예시]

<!-- gridView 의 소스 본문 예시 -->
<w2:gridView visibleRowNum="3" style="height:70px;" defaultCellHeight="24" dataList="data:dlt_books">
    <!-- 중략 -->
</w2:gridView>

4.2보여질 행의 수 지정하기 - 스크립트

원하는 시점에 스크립트를 작성합니다.

GridView의 함수 setVisibleRowNum을 사용합니다.

[소스 코드 예시]

//예제 파일의 스크립트 "scwin.btn_ex1_onclick" 또는 "scwin.btn_ex2_onclick"를 참고하세요.

//GridView 'grd_exam1'의 보여지는 행을 2개로 지정하기
grd_exam1.setVisibleRowNum(2);

/*** 보이는 행을 전체로 지정하기 ***/
var numTotalRow;

//GridView 'grd_exam1'의 전체 행 개수 반환 받기
numTotalRow = grd_exam1.getTotalRow();

//GridView 'grd_exam1'의 보여지는 행을 전체로 지정하기
grd_exam1.setVisibleRowNum(numTotalRow);
