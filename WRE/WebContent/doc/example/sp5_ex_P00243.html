[GridView] 엑셀 다운로드 시 스타일 지정하기 - 공통

1개요

GridView의 엑셀 다운로드(함수 'advancedExcelDownload') 옵션 중 공통으로 스타일을 적용할 수 있는 옵션 설정의 예제입니다.

스타일을 지정할 수 있는 옵션은 아래와 같습니다.
- rowHeight : [default: 없음] Excel 파일로 다운로드 할 때 엑셀의 셀 높이. (단위: pixel)
- displayGridlines : [default: true, false] Excel 파일 전체 셀의 눈금선 제거 유무

P00243

2구현된 기능

  • 엑셀 다운로드 - 기본 동작

  • 엑셀 다운로드 - 공통 스타일 적용

3예제 테스트 방법

다운로드된 엑셀 파일의 행 높이, 눈금선 유무를 비교합니다.

3.1엑셀 다운로드 - 기본 동작

STEP1. 초기 상태 확인하기

서브토탈(subtotal)과 푸터(footer)가 표현되어 있습니다.

[브라우저(Chrome) 실행 예시]

STEP2. 버튼 엑셀 다운로드 - 기본 동작을 클릭합니다.

엑셀 파일 "SP5_EXAMPLE_P00243_TYPE1.xlsx"이 다운로드 됩니다.

STEP3. 실행된 결과를 확인합니다.

다운로드 된 엑셀 파일 'SP5_EXAMPLE_P00243_TYPE1.xlsx'을 실행합니다.

눈금선이 표시되고, 행의 높이가 기본 값으로 설정되어 있습니다.

[다운로드된 엑셀(2021) 파일 예시]

3.2엑셀 다운로드 - 공통 스타일 적용

STEP1. 초기 상태 확인하기

서브토탈(subtotal)과 푸터(footer)가 표현되어 있습니다.

[브라우저(Chrome) 실행 예시]

STEP2. 버튼 엑셀 다운로드 - 공통 스타일 적용을 클릭합니다.

엑셀 파일 "SP5_EXAMPLE_P00243_TYPE2.xlsx"이 다운로드 됩니다.

STEP3. 실행된 결과를 확인합니다.

다운로드 된 엑셀 파일 'SP5_EXAMPLE_P00243_TYPE2.xlsx'을 실행합니다.

눈금선이 표시되지 않고, 행의 높이가 '30px'로 설정되어 있습니다.

[다운로드된 엑셀(2021) 파일 예시]

4구현 예시

4.1엑셀 다운로드 시 행 번호의 스타일 적용하기

원하는 시점에 엑셀 다운로드 스크립트를 작성합니다.

[소스 코드 예시]

//예제 파일의 스크립트 "scwin.btn_ex2_onclick"를 참고하세요.

var jsnOptions;

jsnOptions = {
    fileName: "SP5_EXAMPLE_P00243_TYPE2.xlsx", //엑셀의 파일명
    useSubTotal: "true", //필수 지정 - subTotal 표시
    rowHeight: "30", //Excel 파일로 다운로드 할 때 엑셀의 셀 높이. (단위: pixel)
    displayGridlines: "false" //Excel 파일 전체 셀의 눈금선 제거 유무
};

//GridView "grd_exam1"의 엑셀 다운로드 실행
grd_exam1.advancedExcelDownload(jsnOptions);

5주요 API

advancedExcelDownload( options , infoArr )

options.useSubTotal

options.rowHeight

options.displayGridlines
