[AutoComplete] label, value, delimiter를 이용하여 선택 항목을 표시하기

1개요

AutoComplete의 선택 목록을 Label - Value로 표시하는 예제입니다.
property인 displayMode와 delimiter를 활용합니다.

2구현된 기능

  • Label, Value, delimiter를 이용한 선택 항목 표현

3예제 테스트 방법

3.1delimiter="label delimiter value"

STEP1. 초기 상태 확인하기

AutoComplete의 값을 선택하고 표시된 선택 값 표현이 "Label - Value" 구조임을 확인합니다.

[브라우저(Chrome) 실행 예시]

STEP2. DataList의 Cell을 더블 클릭해서 Edit 모드로 변경 후 값을 변경합니다.

(모바일의 경우 Cell 클릭 후 "SPACE BAR" 키를 사용)

[브라우저(Chrome) 실행 예시]

STEP3. AutoComplete에서 수정한 값이 다시 "Label - Value" 형태로 출력되는 결과를 확인합니다.

[브라우저(Chrome) 실행 예시]

3.2displayModeSync="true"

STEP1. 초기 상태 확인하기

AutoComplete의 값을 선택하고 표시된 선택 값 표현이 "Label - Value" 구조임을 확인하고 선택합니다.

[브라우저(Chrome) 실행 예시]

STEP3. AutoComplete을 다시 선택해 편집 모드에서 "label delimiter value" 형태임을 확인합니다.

[브라우저(Chrome) 실행 예시]

4구현 예시

4.1delimiter 설정하기

AutoComplete을 선택하고 아래 Property를 설정합니다.

displayMode : "label delimiter value" 구조를 선택합니다.

delimiter : delimiter로 설정할 문자열을 입력합니다.

그림 1.웹스퀘어5 SP5 스튜디오의 Property 탭 예시

[소스 코드 예시]

<w2:autoComplete ... displayMode="label delim value" 
                     delimiter=" - ">
    <!-- 내용 생략 -->
</w2:autoComplete>

4.2displayModeSync 설정하기

AutoComplete을 선택하고 아래 Property를 설정합니다.

displayModeSync : "true"로 설정하면 동작합니다.

searchtarget: "both"로 설정해야 displayModeSync가 작동합니다.

그림 2.웹스퀘어5 SP5 스튜디오의 Property 탭 예시

[소스 코드 예시]

<w2:autoComplete delimiter=" - " displayMode="label delim value"
    displayModeSync="true" searchTarget="both">
    <!--내용 생략-->
</w2:autoComplete>
