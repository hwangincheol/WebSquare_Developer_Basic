[GridView] 연속으로 반복되는 셀 데이터의 병합 기능 비교하기

1개요

GridView의 셀의 데이터가 연속으로 반복되는 경우 병합하는 기능을 비교한 예시입니다.
병합 기능을 제공하는 방식은 두 가지로 다음과 같습니다.

  • [바디 컬럼 속성] colMerge : 반복되는 TD의 데이터를 빈 문자열로 할당하고 선을 제거하여 병합 표시. 컬럼 내에서만 동작합니다.

  • [GridView 속성] mergeCells : HTML TD의 속성 'rowspan' 또는 'colspan'을 사용하여 병합 표시.

P00286

2구현된 기능

  • 병합 기능 미사용

  • GridView의 바디 컬럼의 속성 'colMerge'를 사용한 컬럼 내 셀 병합

  • GridView의 속성 'mergeCells'를 사용한 컬럼 내 셀 병합

소스 코드

<!-- gridView 의 소스 본문 예시 -->
<w2:gridView id="grd_exam_1" dataList="data:dlt_books_1" >
    <!-- 중략 -->
    <w2:gBody id="gBody1">
        <w2:row id="row2">
            <w2:column colMerge="true" id="categoryLabel" inputType="text" width="70" displayMode="label" ></w2:column>

    </w2:gBody>
</w2:gridView>

    예시) mergeCol="categoryLabel"
