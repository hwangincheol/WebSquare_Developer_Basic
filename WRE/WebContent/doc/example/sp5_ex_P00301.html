[SelectBox] renderType에 따른 비교

1개요

renderType은 렌더링 되는 구조에 따라 "component", "native", "select" 세 가지 renderType 속성을 지원합니다.
속성에 따라 접근성 지원 방법과 표현 형태의 차이가 존재합니다.
component : 웹 접근성 미지원. 기본 레이아웃.
native : 웹 접근성 지원(title 속성 사용). 브라우저에 무관한 일관적인 디자인.
select : 웹 접근성 지원. 브라우저가 지원하는 form 디자인에 따른 상이한 디자인.
