[FusionChart] 차트 타입을 스크립트로 변경하기 - Scroll Single Series, Scroll Stacked

1개요

Fusionchart의 'Scroll Single Series', 'Scroll Stacked' 타입을 확인할 수 있는 예제입니다.
화면에 구성된 항목를 선택하여 차트의 타입을 변경할 수 있습니다.차트의 타입은 함수 'changeType'를 사용하여 스크립트로 변경할 수 있습니다.

FusionChart 가이드 : https://www.fusioncharts.com/dev/fusioncharts

FusionChart의 타입 목록 : https://www.fusioncharts.com/dev/chart-guide/list-of-charts/

P00310

4구현 예시

4.1스크립트로 'Scroll Single Series Charts' 타입의 차트 변경하기

FusionChart의 함수 'changeType'를 사용하여 구현합니다.

스크립트

// 'Scroll Single Series Charts' 타입 변경 예시)
// Fucionchart의 속성 'seriesColumns' 예시) [['zone1']]

// Fusionchart 'cht_exam1'의 차트 타입을 'Scroll Column 2D'로 변경합니다.
cht_exam1.changeType('scrollcolumn2d');

// Fusionchart 'cht_exam1'의 차트 타입을 'Scroll Bar 2D'로 변경합니다.
cht_exam1.changeType('scrollbar2d');

// Fusionchart 'cht_exam1'의 차트 타입을 'Scroll Line 2D'로 변경합니다.
cht_exam1.changeType('scrollline2d');

// Fusionchart 'cht_exam1'의 차트 타입을 'Scroll Area 2D'로 변경합니다.
cht_exam1.changeType('scrollarea2d');

소스 코드 본문

<w2:fusionchart chartType="ScrollColumn2D" id="cht_exam1" 
    labelNode="type" seriesColumns="[['zone1']]"
    drawType="javascript" ref="data:dlt_chartData_1">
</w2:fusionchart>

4.2스크립트로 'Scroll Stacked Charts' 타입의 차트 변경하기

FusionChart의 함수 'changeType'를 사용하여 구현합니다.

스크립트

// 'Scroll Stacked Charts' 타입 변경 예시)
// Fucionchart의 속성 'seriesColumns' 예시) [['zone1','zone2','zone3']]

// Fusionchart 'cht_exam2'의 차트 타입을 'Scroll Stacked Column 2D'로 변경합니다.
cht_exam2.changeType('scrollstackedcolumn2d');

// Fusionchart 'cht_exam2'의 차트 타입을 'Scroll Stacked Bar 2D'로 변경합니다.
cht_exam2.changeType('scrollstackedbar2d');

소스 코드 본문

<w2:fusionchart chartType="ScrollStackedColumn2D" id="cht_exam2"

    drawType="javascript" ref="data:dlt_chartData_1">
