[컴포넌트 공통] ChooseOption 설정하기

1개요

AutoComplete과 SelectBox의 공통 속성 chooseOption과 chooseOptionLabel 사용 예제입니다.
chooseOption: 선택 안내 메시지 표시 여부를 결정하는 옵션입니다. value는 ""(empty string) 입니다.
chooseOptionLabel: chooseOption을 사용할 때 표시할 텍스트를 설정합니다. 설정하지 않으면 "-선택-"(lable)이 표시됩니다.
chooseOptionValue: 선택 안내 메시지 텍스트의 value (AutoComplete만 해당)

  • AutoComple의 chooseOption, chooseOptionLabel, chooseOptionValue 적용하기

  • SelectBox의 chooseotion, chooseOptionLabel 적용하기

3예제 테스트 방법

3.1chooseOption, chooseOptionLabel, chooseOptionValue 적용 확인하기

STEP 1. chooseOption, chooseOptionLabel 적용을 비교합니다. (AutoComplete, SelectBox 동일)

컴포넌트기본 상태 혹은 클릭해 나타나는 선택 메뉴상단에 선택 메시지 표시를 확인합니다.

그림 1.[브라우저(Chrome) 실행 예시] - AutoComplete

그림 2.[브라우저(Chrome) 실행 예시] - SelectBox

STEP 2. chooseOptionValue 확인하기

chooseOptionValue AutoComplete의 선택 메뉴인 chooseOptionValue를 선택하면 선택 메뉴의 값으로 설정된 "chooseOptionValue" 텍스트가 표시됩니다.

4구현 예시

4.1속성 적용하기

STEP 1. 컴포넌트의 속성을 지정합니다.

  • [필수] chooseOption // [default: "false", true]

  • [필수] chooseOptionLabel // [default:"", string]

  • [필수] chooseOptionValue // [default: "", string] (AutoComplete만)

그림 4.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

[소스 코드 예시]

<!-- AutoComplete 컴포넌트의 사용 예시입니다. -->

<w2:autoComplete ... chooseOption="true" chooseOptionLabel="chooseOptionValue"
chooseOptionValue="chooseOptionValue">
<!-- 중략 -->
</w2:autoComplete>
