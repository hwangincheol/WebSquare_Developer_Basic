[GridView] 읽기 전용으로 설정된 특정 컬럼의 데이터 붙여넣기(Ctrl+V) 허용하기

1개요

읽기 전용으로 설정된 특정 컬럼의 셀에 데이터 붙여넣기(CTRL+V)를 허용하는 예제입니다.

P00343

2구현된 기능

  • 읽기 전용으로 설정된 특정 컬럼의 데이터 붙여넣기(Ctrl+V) 허용하기

3예제 테스트 방법

3.1읽기 전용으로 설정된 특정 컬럼의 데이터 붙여넣기(Ctrl+V) 허용하기

STEP 1. 초기 상태를 확인합니다.

문자열을 복사할 Input 컴포넌트와 붙여넣기(Ctrl+V)를 테스트할 GridView가 구성되어 있습니다.

그림 1.브라우저(Chrome) 실행 예시 - 문자열 복사를 위한 Input

그림 2.브라우저(Chrome) 실행 예시 - 문자열 붙여넣기 테스트를 위한 GridView

STEP 2. Input의 문자열을 복사합니다.

Input에 할당된 문자열을 복사(Ctrl+C)합니다.

STEP 3. GridView의 컬럼 '붙여넣기 불가'의 셀에 붙여넣기(Ctrl+V)합니다.

GridView의 컬럼 '붙여넣기 불가'의 2번째 로우의 셀을 클릭하고 키보드 키 입력(Ctrl+V)을 통해 붙여넣기합니다.

그림 3.브라우저(Chrome) 실행 예시

STEP 4. 실행된 결과를 확인합니다.

붙여넣기가 실행되지 않습니다.

STEP 5. GridView의 컬럼 '붙여넣기 가능'의 셀에 붙여넣기(Ctrl+V)합니다.

GridView의 컬럼 '붙여넣기 가능'의 2번째 로우의 셀을 클릭하고 키보드 키 입력(Ctrl+V)을 통해 붙여넣기합니다.

그림 4.브라우저(Chrome) 실행 예시

STEP 6. 실행된 결과를 확인합니다.

데이터가 'NOW'로 변경됩니다.

그림 5.브라우저(Chrome) 실행 예시

4구현 예시

4.1읽기 전용으로 설정된 특정 컬럼의 데이터 붙여넣기(Ctrl+V) 허용하기

STEP1. GridView의 속성을 정의합니다.

  • [필수] readOnly="true"

    전체 셀을 읽기 전용으로 적용합니다. 또는 컬럼, 셀 단위로 읽기 전용을 적용합니다.

  • [필수] checkReadOnlyOnPaste="true"

    읽기 전용 셀인 경우 붙여넣기를 금지합니다.

  • [필수] checkReadOnlyOnPasteEnable="컬럼의 ID"

    읽기 전용 컬럼 중 붙여넣기를 허용할 컬럼의 ID를 지정합니다. 여러 개의 ID를 지정하는 경우 구분자 ','를 사용합니다.

    예시 1) 단건의 ID 적용

    checkReadOnlyOnPasteEnable="col_2"

    예시 2) 다건의 ID 적용

    checkReadOnlyOnPasteEnable="col_2,col_3"

그림 6.웹스퀘어5 SP5 스튜디오의 DataList Property View(속성창) 예시

소스 코드

<w2:gridView readOnly="true" checkReadOnlyOnPaste="true" checkReadOnlyOnPasteEnable="col_2">
    <!-- 중략 -->
    <w2:gBody style="" id="gBody1">
        <w2:row style="" id="row2">
            <w2:column id="col_1" width="110" inputType="text" displayMode="label"></w2:column>
            <w2:column id="col_2" width="110" inputType="text" displayMode="label"></w2:column>
        </w2:row>
    </w2:gBody>
</w2:gridView>
