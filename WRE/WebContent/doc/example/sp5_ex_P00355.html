[Input] 입력 필드에 'readOnly' 또는 'disabeld'가 설정된 경우에도 힌트 문자열(placeholder) 표시하기

1개요

Input의 속성 'showPlaceHolderOnReadOnly' 설정 값 비교 예제입니다.
이 속성은 속성 'placeholder'에 문자열이 설정되어있고, 속성 'readOnly'의 설정 값이 'true'이거나 속성 'disabled'의 설정 값이 'true'로 설정된 경우에 동작합니다.
속성의 설정 값에 따른 동작은 다음과 같습니다.

  • true : 속성 'placeholder'의 설정 값이 입력 필드에 표시됩니다.

  • false : (기본 값) 속성 'placeholder'의 설정 값이 입력 필드에 표시되지 않습니다.함수 'setPlaceholder'는 속성 'placeholder'의 값을 변경하는 기능을 제공합니다.

P00355

속성 'readOnly'의 설정 값이 'true'이거나 속성 'disabled'의 설정 값이 'true'로 설정된 경우, 'placeholder' 문자열이 표시되도록 설정할 때 속성 'showPlaceHolderOnReadOnly'의 설정 값을 'true'로 지정합니다.

2구현된 기능

  • 속성 showPlaceHolderOnReadOnly의 설정 값을 false로 지정

  • 속성 showPlaceHolderOnReadOnly의 설정 값을 true로 지정

3예제 테스트 방법

3.1속성 showPlaceHolderOnReadOnly의 설정 값을 false로 지정

STEP 1. 초기 상태를 확인합니다.

영역 [(기본 설정) showPlaceHolderOnReadOnly="false"]에 구성된 Input을 확인합니다.
두 개의 Input은 속성 'placeholder'의 설정 값이 '입력 힌트 예시입니다.'으로 지정되어 있고, 속성 'showPlaceHolderOnReadOnly'의 설정 값이 'false'로 지정된 상태입니다.
첫 번째 Input은 속성 'readOnly'가 설정되고 있고, 두 번째 Input은 속성 'disabled '가 설정되어 있습니다.

STEP 2. 실행된 결과를 확인합니다.

두 개의 Input에 속성 'placeholder'에 지정한 값이 표시되지 않습니다.

그림 1.브라우저(Chrome) 실행 예시

3.2속성 showPlaceHolderOnReadOnly의 설정 값을 true로 지정

STEP 1. 초기 상태를 확인합니다.

영역 [(기본 설정) showPlaceHolderOnReadOnly="true"]에 구성된 Input을 확인합니다.
두 개의 Input은 속성 'placeholder'의 설정 값이 '입력 힌트 예시입니다.'으로 지정되어 있고, 속성 'showPlaceHolderOnReadOnly'의 설정 값이 'true'로 지정된 상태입니다.
첫 번째 Input은 속성 'readOnly'가 설정되고 있고, 두 번째 Input은 속성 'disabled '가 설정되어 있습니다.

STEP 2. 실행된 결과를 확인합니다.

두 개의 Input에 속성 'placeholder'에 지정한 값이 표시됩니다.

그림 2.브라우저(Chrome) 실행 예시

Input의 속성을 정의합니다.

  • [필수] showPlaceHolderOnReadOnly="true"

    (옵션 설명)

    false : [default] readOnly="true" 또는 disabled="true"인 경우, placeholder 속성으로 지정된 값을 표시하지 않음

    true : readOnly="true" 또는 disabled="true"인 경우, placeholder 속성으로 지정된 값을 표시

  • [필수] placeholder="문자열"

    예시) placeholder="입력 힌트 예시입니다."
