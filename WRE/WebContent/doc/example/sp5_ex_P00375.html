[InputCalendar] 브라우저에 랜더링할 때, 입력 필드와 달력 아이콘 영역을 태그 'div' 또는 'table'로 구성하기

1개요

InputCalendar의 속성 'renderDiv'의 설정 값에 따른 기능을 비교하는 예제입니다.
이 속성은 컴포넌트의 입력 필드와 달력 아이콘 영역을 브라우저에 랜더링할 때 'Div' 태그 또는 'Table' 태그로 생성할지를 설정할 수 있습니다.
설정 값에 따른 구성은 다음과 같습니다.

  • true: [default] 입력 필드와 달력 아이콘 영역을 'div'로 구성되며 달력 아이콘이 'button'으로 구성됩니다.

  • false: 입력 필드와 달력 아이콘 영역을 'table'로 구성되며 달력 아이콘이 'img'으로 구성됩니다.

2구현된 기능

  • 입력 필드와 아이콘 영역을 태그 'div'로 구성하기

  • 입력 필드와 아이콘 영역을 태그 'table'로 구성하기

3예제 테스트 방법

3.1입력 필드와 아이콘 영역을 태그 'div'로 구성하기

STEP 1. 실행된 결과를 확인합니다.

예제 영역 [(기본 설정 값) 입력 필드와 아이콘 영역을 태그 'div'로 구성하기]에 구성된 InputCalendar를 브라우저 개발자 도구의 Elements(요소)탭을 통해 확인합니다.

입력 필드와 달력 아이콘 영역을 'div'로 구성되며 달력 아이콘이 'button'으로 구성됩니다.

태그 'div'로 구성된 InputCalendar에 별도의 CSS가 설정되어 달력 아이콘의 이미지와 위치가 다릅니다. (예제 사이트 공통 적용)

그림 1.브라우저(Chrome) 실행 예시

브라우저에 랜더링된 HTML 구조

<div id="mf_wq_uuid_16" style="width:110px;height: 26px;"
  class="w2inputCalendar_div w2inputCalendar_type_yearMonthDate">
  <div id="mf_wq_uuid_16_div_input" class="w2inputCalendar_div_input" style="width: 84px; height: 26px;">
    <input id="mf_wq_uuid_16_input" name="mf_wq_uuid_16_input" class="w2inputCalendar_divInput " tabindex="0"
      style="width: 50px; height: 24px;">
  </div>
  <div id="mf_wq_uuid_16_div_img" class="w2inputCalendar_div_img">
    <button type="button" id="mf_wq_uuid_16_img" class="w2inputCalendar_button" tabindex="0">달력에서 선택</button>
  </div>
</div>

브라우저에 랜더링된 HTML Elements의 'id'는 실행 시점에 동적으로 부여되어 환경에 따라 다릅니다.
