[Textarea] CSS로 리사이즈 기능 적용하기

1개요

Textarea의 리사이즈 기능을 CSS로 적용하는 예제입니다.
이 기능은 CSS 속성 'resize'를 사용하여 구현할 수 있습니다.
리사이즈 기능이 적용되면 Textarea의 우측 하단에 아이콘이 표시되며 마우스를 드래그하여 크기를 조절할 수 있습니다.

P00407

웹스퀘어에서 제공하는 기능이 아니며 브라우저의 지원 여부에 따라 동작됩니다.

다음은 브라우저 지원 버전입니다. (2023.11.01 기준 작성)

- Chrome: 4

- Edge: 79

- Firefox: 5

- Opera: 15

- Safari: 4

2구현된 기능

  • 리사이즈 미적용

  • 리사이즈 상하, 좌우 적용

  • 리사이즈 좌우 적용

  • 리사이즈 상하 적용

3예제 테스트 방법

3.1리사이즈 미적용

STEP 1. 초기 상태를 확인합니다.

예제 영역 [리사이즈 미적용]의 Textarea를 확인합니다.

STEP 2. 실행 결과를 확인합니다.

우측 하단에 리사이즈 아이콘이 없으며, 사용자가 마우스 드래그를 이용해 리사이즈 할 수 없습니다.

그림 1.브라우저(Chrome) 실행 예시

4구현 예시

4.1CSS로 리사이즈 기능 설정하기

STEP1. CSS를 정의합니다.

CSS - 프로젝트 전체에 적용하는 경우

/* 프로젝트 전체에 적용하는 경우 아래와 같이 정의합니다. */

/* 웹스퀘어 Textarea에 적용된 class selector 사용 */
.w2textarea { resize: both; }

/* 또는 */
textarea { resize: both; }

CSS - 컴포넌트 단위로 적용하는 경우

/* 이 예제 파일에서 사용한 class는 다음의 경로에서 확인할 수 있습니다. */
/* [프로젝트 경로]/WebContent/css/example.css */

/* P00407.xml Textarea의 리사이즈 예제 기능 비교 */
.P00407_resize_none { resize: none; } /* resize 사용 안함 */
.P00407_resize_both { resize: both; } /* resize 상하, 좌우 사용 */
.P00407_resize_horizontal { resize: horizontal; } /* resize 좌우 사용 */
.P00407_resize_vertical { resize: vertical; } /* resize 상하 사용 */

STEP2. 속성 'class'를 지정합니다.

컴포넌트 단위로 적용하는 경우 Textarea의 속성 'class'에 CSS class명을 지정합니다.

  • [필수] class="class명"

    (예시)

    class="P00407_resize_both"
