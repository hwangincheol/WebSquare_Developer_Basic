[FloatingLayer] 헤더 영역에 타이틀 설정하기

1개요

헤더 영역의 타이틀을 설정하는 예제입니다.
이 기능은 속성 'title'과 함수 'setTitle'로 구현할 수 있습니다.

P00412

이 예제는 닫기 버튼이 CSS로 감춰진 상태입니다.

2구현된 기능

  • 타이틀 설정하지 않기

  • 타이틀 설정하기

3예제 테스트 방법

3.1타이틀 설정하지 않기

STEP 1. 실행된 결과를 확인합니다.

예제 영역 [(기본 설정) 헤더 영역에 타이틀이 설정되지 않은 상태]의 'FloatingLayer'를 확인합니다.
헤더 영역에 타이틀이 표시되지 않습니다.

그림 1.브라우저(Chrome) 실행 예시

3.2타이틀 설정하기

STEP 1. 실행된 결과를 확인합니다.

예제 영역 [헤더 영역에 타이틀이 설정된 상태]의 'FloatingLayer'를 확인합니다.
헤더 영역에 타이틀이 '타이틀 지정 예시입니다'로 표시되어 있습니다.

그림 2.브라우저(Chrome) 실행 예시

STEP 2. 타이틀을 변경합니다.

버튼 타이틀 변경하기를 클릭합니다.

STEP 3. 실행된 결과를 확인합니다.

헤더 영역에 타이틀이 '타이틀이 변경되었습니다'로 변경됩니다.

그림 3.브라우저(Chrome) 실행 예시

4구현 예시

4.1속성으로 타이틀 설정하기

속성을 정의합니다.

  • [필수] title="타이틀 문자열"

    예시) title="타이틀 지정 예시입니다"

4.2스크립트로 타이틀 설정하기

FloatingLayer의 함수 'setTitle'를 이용하여 스크립트를 작성합니다.
세부 지정은 아래의 스크립트 예시에 작성되어 있습니다.

스크립트

//예제 파일에서는 스크립트 scwin.btn_exam2_1_onclick에 작성되어 있습니다.

// FloatingLayer 'flt_exam2'의 타이틀을 변경합니다.
flt_exam2.setTitle("타이틀이 변경되었습니다");

5주요 API

  • title

  • setTitle( title )
