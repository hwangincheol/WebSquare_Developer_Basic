[PageList] 브라우저에 랜더링할 때, 'table' 또는 'ul' 태그로 구성하기

1개요

속성 'renderType'의 설정 값에 따른 기능을 비교하는 예제입니다.
이 속성은 브라우저에 랜더링하기 위해 컴포넌트를 구성할 때 사용할 HTML 요소를 선택할 수 있습니다.
설정 값에 따른 구성은 다음과 같습니다.

  • "table": [default] 버튼과 페이지 숫자 영역을 'table'로 구성합니다.

  • "list": 버튼과 페이지 숫자 영역을 'ul', 'li'로 구성합니다.

P00416

2구현된 기능

  • 'table'로 구성하기

  • 'ul', 'li'로 구성하기

3예제 테스트 방법

3.1'table'로 구성하기

STEP 1. 실행된 결과를 확인합니다.

예제 영역 [(기본 설정) 'table'로 구성하기]의 'PageList'를 브라우저의 개발자 도구(Elements)로 확인합니다.

그림 1.브라우저(Chrome) 실행 예시

브라우저에 랜더링된 HTML 구조

<!-- 주요 속성만 작성되었습니다. -->
<div class="w2pageList w2pageList_pagePrev_last w2pageList_prev_last ">
  <table class="w2pageList_table">
    <tbody>
      <tr>
        <td role="button" class="w2pageList_control_pagePrev w2pageList_col_prevPage"
          title="이전 목록 페이지"></td>
        <td role="button" class="w2pageList_control_prev w2pageList_col_prev" 
          title="이전 페이지"></td>
        <td class="w2pageList_col_label">
          <div class="w2pageList_control_label w2pageList_label_selected" index="1"
            role="button" title="1 선택">1</div>
        </td>
        <!-- 중략 -->
        <td class="w2pageList_col_label">
          <div class="w2pageList_control_label w2pageList_label" index="10" role="button"
            title="10">10</div>
        </td>
        <td role="button" class="w2pageList_control_next w2pageList_col_next" 
          title="다음 페이지"></td>
        <td role="button" class="w2pageList_control_pageNext w2pageList_col_nextPage"
          title="다음 목록 페이지"></td>
      </tr>
    </tbody>
  </table>
</div>
