[AutoComplete] 편집 모드 전환 시 입력 필드의 문자열 선택 유형 설정하기

1개요

속성 'editType'의 설정 값 비교 예제입니다.
이 속성은 편집 모드 전환 시 입력 필드의 문자열 선택 유형을 설정할 수 있습니다.
설정 값에 따른 동작은 다음과 같습니다.

  • "focus" : 편집 모드 전환 시 입력 필드의 문자열 마지막에 커서가 위치됩니다.

  • "select" : [default] 편집 모드 전환 시 입력 필드의 문자열이 선택됩니다.

2구현된 기능

  • 편집 모드 전환 시 입력 필드의 문자열 선택하기

3예제 테스트 방법

3.1편집 모드 전환 시 입력 필드의 문자열 마지막에 커서 위치시키기

STEP1. 초기 상태 확인하기

영역 [editType="focus"]에 구성된 예제를 확인합니다.

STEP2. 편집 모드 전환하기

'AutoComplete'을 클릭합니다. 우측에 구성된 버튼을 클릭하면 목록이 표시됩니다. 버튼을 제외한 컴포넌트 영역을 클릭합니다.
