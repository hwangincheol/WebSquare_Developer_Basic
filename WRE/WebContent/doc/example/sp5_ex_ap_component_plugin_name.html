부록 A.컴포넌트의 Plugin Name

컴포넌트명(Plugin Name)입니다.
주로 공통 스크립트에서 불특정 컴포넌트를 받아 타입별로 분기하여 로직을 구성할 때 사용되거나, 
동적으로 컴포넌트를 생성($p.dynamicCreate)할 때 사용됩니다.
